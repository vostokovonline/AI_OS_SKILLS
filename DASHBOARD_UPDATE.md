# Dashboard Update - Goal System Integration

## Новые возможности дашборда

### 1. Отображение типов целей
Каждая цель теперь показывает свой тип с иконкой:
- 🎯 **Achievable** (Выполнимая) - можно завершить
- 🔄 **Continuous** (Непрерывная) - постоянное улучшение
- 🧭 **Directional** (Векторная) - направление, не выполняется
- 🔍 **Exploratory** (Исследовательская) - поиск знаний
- ⚙️ **Meta** (Мета-цель) - улучшение системы

### 2. Уровни глубины
Показывает уровень в иерархии:
- **L0: Mission** - намерение
- **L1: Strategic** - стратегические подцели
- **L2: Operational** - операционные подцели
- **L3: Tactical/Atomic** - атомарные цели (⚛️)

### 3. Новые кнопки управления
- ⏸️ **Пауза** - приостановить цель
- ▶️ **Запуск** - выполнить цель
- 🔻 **Декомпозировать** - разбить на подцели
- ✓ **Оценить** - проверить выполнение
- 🗑️ **Удалить** - удалить цель (с подтверждением)
- 📋 **Подробнее** - детальная информация

### 4. Фильтрация и сортировка
**Фильтры:**
- По статусу (done, active, pending, blocked, improving)
- По типу цели (доступно для новых целей)

**Сортировка:**
- Дата создания (новые/старые)
- Прогресс
- Статус
- Тип цели
- Уровень глубины

### 5. Расширенная статистика
**Общая статистика:**
- Всего целей
- Выполнено
- Активных
- Средний прогресс

**По типам целей:**
- 🎯 Achievable - количество
- 🔄 Continuous - количество
- 🧭 Directional - количество
- 🔍 Exploratory - количество
- ⚙️ Meta - количество

**По уровням:**
- L0: Mission - количество
- L1: Strategic - количество
- L2: Operational - количество
- L3: Tactical/Atomic - количество

**Атомарные цели:**
- ⚛️ Показывает количество целей готовых для Task Layer

### 6. Детали цели
При нажатии "📋 Подробнее" показывается:
- **ID, статус, прогресс, дата обновления**
- **Тип цели** (с иконкой и названием)
- **Атомарная** (Да/Нет)
- **Уровень глубины** (L0-L3)
- **Родительская цель** (если есть)
- **Описание**
- **Домены** (теги)
- **Критерии завершения** (если есть)
- **Результат оценки** (если была оценка)
  - Тренд для continuous целей
  - Пройдена/Оценка
- **Логи выполнения** (агенты, результаты, ошибки)

## Workflow работы с целями

### Создать цель
Цель автоматически классифицируется при создании.

### Декомпозировать цель
1. Нажмите 🔻 на цели
2. Система создаст 3-7 подцелей по доменам
3. Каждая подцель будет на уровень глубже

### Выполнить атомарную цель
1. Декомпозируйте до L3 (Tactical/Atomic)
2. Появится метка ⚛️
3. Нажмите ▶️ для выполнения
4. Goal executor выполнит её через агентов

### Оценить выполнение
1. После выполнения нажмите ✓
2. Система проверит критерии успеха
3. Если пройдена - может сгенерировать следующую цель

## API для работы с целями

Все эндпоинты доступны через `http://ns_core:8000`:

```bash
# Создать цель
POST /goals/create
{
  "title": "Название цели",
  "description": "Описание",
  "auto_execute": false
}

# Классифицировать цель
POST /goals/classify
{
  "title": "Название",
  "description": "Описание"
}

# Декомпозировать
POST /goals/{goal_id}/decompose?max_depth=3

# Оценить выполнение
POST /goals/{goal_id}/evaluate

# Получить дерево целей
GET /goals/{goal_id}/tree

# Статистика
GET /goals/stats
```

## Демонстрация типологии

### Пример: Achievable цель
```
🎯 Настроить монетизацию | L1 | 45%
🏷️ domains: бизнес, инфраструктура
```
→ Можно декомпозировать → выполнить → оценить

### Пример: Continuous цель
```
🔄 Улучшать состояние пользователя | L0 | 67%
```
→ Никогда не завершается, проверяется тренд

### Пример: Directional цель
```
🧭 Оставить след в истории | L0 | 0%
```
→ Не декомпозируется, задает направление

### Пример: Atomic цель
```
⚛️ � Достичь уровня Mg | L3 | 100%
```
→ Готова для Task Layer, можно превращать в задачи

## Визуальные индикаторы

**В строке цели:**
- 🎯🔥🔍 - иконка типа
- ⚛️ - атомарная цель
- L0-L3 - уровень глубины
- 0-100% - прогресс
- 🏷️ - домены (под строкой)

**Кнопки:**
- 6 кнопок управления
- Кнопка 🔻 появляется для всех не-atomic целей
- Кнопка ✓ для оценки выполненных целей
