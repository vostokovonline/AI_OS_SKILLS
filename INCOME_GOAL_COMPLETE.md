### Автоматическое выполнение

Система теперь будет:

1. **Автоматически выполнять** L3 atomic цели (14 целей)
2. **Адаптироваться** к ошибкам через MemorySignal
3. **Декомпозировать** L2 цели при необходимости
4. **Обновлять прогресс** L1 и L0 целей

### Пример адаптации:

```
Цикл 1: "Написать код MVP" → technical failure
  ↓ MemorySignal: recent_failure
  ↓ Следующий цикл: система избегает похожие задачи
  ↓ Через 5 циклов: сигнал забывается (TTL=0)

```
